# üçΩ Dish Dash v4.3

Smart Dishwasher Chore Tracker - ESP32 Fingerprint Game

## Features
- **Fingerprint Recognition** (AS608) - scan finger to score points
- **LED Matrix Display** (MAX7219 4x) - scrolling text with German umlauts
- **Motion Detection** (SR602 PIR) - auto display on/off
- **Sound Effects** (PAM8403 + MOSFET) - Mario-inspired melodies
- **Web Dashboard** - full control from any phone/browser
- **OTA Updates** - update over WiFi from this repo

## Hardware

## Files
- `app.py` - Main source code (MicroPython)
- `dashboard.html` - Web UI
- `main.py` - Boot loader
- `version.json` - Version info for OTA
- `dist/` - Compiled files (auto-generated by GitHub Actions)

## OTA Update
The `dist/` folder is automatically built by GitHub Actions when you push changes.
Each Dish Dash device can check for updates via Dashboard ‚Üí Config ‚Üí Update.

## First Setup
1. Flash MicroPython 1.27 to ESP32
2. Upload `main.py`, `app.mpy` (from dist/), and `dashboard.html`
3. Power on ‚Üí Connect to "DISH-DASH-Setup" WiFi
4. Open 192.168.4.1 ‚Üí Enter home WiFi credentials
5. Device reboots and connects to your network
